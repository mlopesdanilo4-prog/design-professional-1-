Sobre esta disciplina
Esta pasta serve como ponto de referência para o repositório oficial da disciplina Algoritmos e Pensamento Computacional , onde foram desenvolvidos projetos usando C focados em lógica e cálculo computacional.

Conteúdos abordados
Lógica de programação e estruturas de controle (if, switch, loops)
Manipulação de dados com arrays, funções e structs
Desenvolvimento de pequenos programas e exercícios práticos em C
