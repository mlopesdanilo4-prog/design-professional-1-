Sobre esta disciplina
Esta pasta existe como ponte para o repositório oficial da disciplina Modelagem de Banco de Dados , que reúne as entregas envolvendo DER, MER, normalização e o projeto semestral de banco de dados.

Conteúdos abordados
Modelagem Conceitual, Lógica e Física de um projeto do semestre
Entidades, relacionamentos, cardinalidade e especialização
Normalização (1FN, 2FN, 3FN) e criação de esquemas relacionais
